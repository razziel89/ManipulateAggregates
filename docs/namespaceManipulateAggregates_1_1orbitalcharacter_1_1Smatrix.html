Detailed Description

Compute overlaps of basis functions and molecular orbitals.

Taken from http://www.mathematica-journal.com/2012/02/evaluation-of-gaussian-molecular-integrals/ and altered a bit before transforming to Python code. All thanks goes to the authors.

Normalize a molecular orbital defined by some coefficients to the given occupation.
