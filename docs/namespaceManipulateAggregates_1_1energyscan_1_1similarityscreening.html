Detailed Description

Screen aggregate geometries with respect to similarity.

This subsubmodule is part of ManipulateAggregates.energyscan. It implements the third step of the 3-step procedure that creates low energy aggregate geometries.

Parallelization is only supported for the determination of pointgroups.
